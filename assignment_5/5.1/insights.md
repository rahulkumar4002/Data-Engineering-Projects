# Brief Insights

- Successfully created and processed a Spark DataFrame using PySpark.

- Missing values in the `Age`, `Department`, and `Salary` columns were identified and replaced using appropriate default values.

- Duplicate records were removed to improve data quality and avoid incorrect analytical results.

- Filtering operations were performed to retrieve specific employee records based on age and department.

- Department-wise aggregations such as average salary, total salary, and employee count were calculated using `groupBy()` and aggregation functions.

- A new derived column (`Bonus`) was created to demonstrate column transformation capabilities.

- The `Salary` column was renamed to `Monthly_Salary` for better readability and schema consistency.

- The final cleaned dataset was sorted and prepared for further analysis and reporting.

- This project demonstrates essential data preprocessing techniques commonly used in data engineering and big data applications.
