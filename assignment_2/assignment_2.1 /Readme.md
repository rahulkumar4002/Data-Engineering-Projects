#Sales Data Analysis Using SQL

## Project Overview

This project analyzes the Superstore Sales Dataset using SQL. The objective is to perform data exploration, filtering, aggregation, sorting, and business-oriented analysis to extract meaningful insights from sales data.

**Tools Used:** MySQL Workbench, SQL

**Dataset:** Superstore Sales Dataset

---

## Key Objectives

- Explore and understand the dataset
- Apply filtering techniques using WHERE clause
- Perform aggregations using SQL functions
- Analyze sales performance by category and region
- Identify top-performing products and customers
- Study monthly sales and profit trends
- Validate data quality through duplicate and null checks

---

## Key Findings

### Sales Performance
- The dataset contains sales transactions across multiple regions and product categories.
- Revenue generation is concentrated among a limited number of products.
- Sales performance varies significantly across customer segments.

### Category Analysis
- Product categories contribute differently to overall revenue and profitability.
- Some categories generate high sales but comparatively lower profit margins.
- Category-level analysis helps identify business growth opportunities.

### Customer Analysis
- A small group of customers contributes a significant share of total sales.
- Top customers play an important role in overall revenue generation.
- Customer retention strategies can improve long-term profitability.

### Regional Analysis
- Sales performance differs across geographical regions.
- Certain regions consistently contribute higher revenue.
- Regional analysis supports targeted business planning.

### Trend Analysis
- Monthly sales and profit trends reveal seasonal business patterns.
- Demand fluctuations can be observed throughout the year.
- Trend analysis supports forecasting and inventory planning.

### Data Quality Validation
- Duplicate order checks were performed.
- Missing value checks were conducted.
- Data quality validation ensures reliable business reporting.

---

## Conclusion

The Superstore dataset was successfully analyzed using SQL. Various techniques such as filtering, aggregation, grouping, sorting, and trend analysis were applied to generate business insights. The analysis identified key sales drivers, top-performing customers, category performance, regional trends, and monthly business patterns. The results demonstrate how SQL can be used effectively for data analysis and business decision-making.
