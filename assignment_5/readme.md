
Data Cleaning and Transformation using PySpark
📌 Project Overview

This project demonstrates data cleaning and transformation operations using Apache Spark (PySpark). The dataset contains duplicate records and missing values which are processed to improve data quality and generate meaningful insights.

The project focuses on understanding Spark DataFrames, transformations, aggregations, and data preprocessing techniques commonly used in Data Engineering.

🚀 Objectives
Understand Spark DataFrames and immutability
Handle missing (null) values
Remove duplicate records
Apply filtering conditions
Perform aggregations using groupBy()
Calculate count, sum, average, minimum, and maximum values
Rename and cast columns
Understand shuffle operations and wide transformations
Build a complete data processing pipeline
🛠️ Technologies Used
Python
Apache Spark (PySpark)

⚙️ Operations Performed
Loaded data into Spark DataFrame
Inspected schema using printSchema()
Removed duplicate records using dropDuplicates()
Handled null values using fillna()
Applied filtering conditions
Performed aggregations using groupBy()
Calculated total, average, minimum and maximum values
Renamed columns

Built a complete data processing pipeline
