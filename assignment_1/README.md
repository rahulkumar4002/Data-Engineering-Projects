# Data Exploration and Cleaning Using Pandas

## Project Overview
This project demonstrates fundamental data exploration and data cleaning operations using the Python Pandas library. The primary goal of this assignment is to preprocess raw CSV data by identifying missing values, removing duplicates, performing filtering operations, creating derived columns, and exporting a cleaned dataset for further analysis.

This project reflects essential data handling techniques commonly used in Data Analytics and Data Engineering workflows.

---

# Project Objectives

- Load and inspect CSV datasets
- Explore dataset structure and statistics
- Handle missing values effectively
- Remove duplicate records
- Perform data filtering operations
- Create derived/engineered columns
- Export cleaned data for downstream processing
